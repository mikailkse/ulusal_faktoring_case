# Flutter - Mobile Case ULUSAL FAKTORING

## İlk Çalıştırma

-**Dev ortamı için**

    flutter run --target=lib/main_dev.dart --flavor=dev

-**Prod ortamı için**

    flutter run --target=lib/main_prod.dart --flavor=prod

kodlarını girerek ilk çalıştırmayı sağlayabilirsiniz.
